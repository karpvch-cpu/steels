# Changelog

## [0.3.0] — 2026-02-20

### Добавлено
- **Механические свойства** для всех 31 марки стали
  - Предел текучести (σ₀.₂), временное сопротивление (σᵥ), относительное удлинение (δ₅)
  - Данные по диапазонам толщин из стандартов
  - Ударная вязкость (KCU/KV) с температурой испытания
  - Свариваемость + CEV (углеродный эквивалент)
- Тултипы с пояснениями мех. свойств простым языком
- Мех. свойства в таблице сравнения с подсветкой лучшего значения
- Перетаскиваемый разделитель между панелями (resizer)
- Поиск перенесён в левую панель (над списком марок)
- Динамический счётчик в шапке: кол-во марок · стандартов · стран

### Изменено
- Шапка упрощена: логотип + статистика + настройки
- Убрана кнопка CSV-экспорта (временно)

## [0.2.0] — 2026-02-19

### Добавлено
- Тёмная тема (18 цветовых токенов)
- 3 уровня масштабирования: S (90%) / M (100%) / L (115%)
- Searchable теги для кросс-стандартного поиска
- Виды продукции для каждой марки
- Экспорт CSV (позже убран)

## [0.1.0] — 2026-02-18

### Начальная версия
- 31 марка стали: ГОСТ, EN, ASTM, JIS, GB/T
- Химический состав из стандартов
- Аналоги между стандартами
- Поиск по названию, стандарту, категории
- Поиск по хим. составу (фильтр по элементам)
- Сравнение до 10 марок бок-о-бок
- Карточка с полным составом
